# Algorithms and Data Structures Practice

## Description
This repository was created for study purposes. It contains solved algorithm and data structure problems from HackerRank, LeetCode, and similar platforms.

## Motivation
The goal of this repository is to improve my problem-solving skills and English comprehension while deepening my understanding of algorithms and data structures through consistent practice.  

## Rules for this study

**First:**  
AI cannot be used for code review or bug fixing while I am solving the problem.

**Second:**  
All the information necessary to solve the problem must be searched in official documentation, reliable websites, or books. That information must be in English.

**Third:**  
The time limit to complete each challenge is one week.

**Fourth:**  
The problem must be solved only in the platform code editor. No code assistants are allowed.
